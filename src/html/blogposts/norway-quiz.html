Process

I used ChatGPT to generate quiz questions about Norway, then carefully filtered and double-checked the answers for accuracy. The project had a quick turnaround, so I focused on creating a user-friendly and engaging experience.

Code

The coding process involved continuous improvement. While exploring VS Code, I discovered the "Refactor" button, which may have led to some over-abstraction in certain code sections. However, the core logic functions well and withstood testing. Ideally, the submit/review logic wouldn't be called from within another loop, but this is an area for future refinement.

Design

I aimed for a minimalist and modern design. After exploring various sources, I drew inspiration primarily from Brilliant.org and designs by Paresh Khatri on Dribbble.
