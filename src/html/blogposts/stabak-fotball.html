This webpage displays the players for my local fotball team Stabæk. This project was a induvidual student assignment for Høyskolen Kristiania.

Goals

The goal for this project was to use a flexbox or grid system to create a sports-team realted site. Each Card in the layout was to feature a couple of facts about the player. I choose to create a close copy of the orginal website of stabak.no.
